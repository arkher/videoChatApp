# Video-Group-Meeting

A simple video chat application for multi-users based on React, Node Express and WebRTC.

**Technologies Used**

- Vite js
- React
- Node + Express
- WebRTC
- Socket.io
- [Syled-components](https://styled-components.com/)
- [Simple-peer](https://github.com/feross/simple-peer)

---

## Features

- Join a room
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing

## Installation
### Clone
- Clone this repo to your local machine using `git clone https://github.com/arkher/videoChatApp`

### Setup
**Client**
After moving to server folder:
To install dependencies: ```yarn```
To run: ```yarn dev```

**Server**
After moving to server folder:
To install dependencies: ```yarn```
To run: ```yarn start```
